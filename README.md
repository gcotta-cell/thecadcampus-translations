# The CAD Campus - Translation Framework

This repository contains all language files used by The CAD Campus website.

## Purpose

The goal of this repository is to provide a centralized translation system that allows the website to support multiple languages while maintaining a single Thinkific site.

## Supported Languages

- English
- Español
- Português
- Deutsch
- Français
- Русский
- 中文
- 日本語
- 한국어
- हिन्दी
- Türkçe

## Repository Structure

```
languages/
    en.json
    es.json
    pt.json
    de.json
    fr.json
    ru.json
    zh.json
    ja.json
    ko.json
    hi.json
    tr.json

tcc-language-loader.js
```

English (`en.json`) is the master language. All other translations are derived from it to preserve the voice and tone of The CAD Campus.
